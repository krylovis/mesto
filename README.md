# Проект: Место

Собрать проект: npm run build
Запустить проект: npm run dev

**Описание проекта**

Проект для добавления фото. 
С возвожностью редактирование имени и профессии автора фотографий.

**Технологии**

* HTML
* CSS
* Flexbox
* Grid Layout
* JavaScript
